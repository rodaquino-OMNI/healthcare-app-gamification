# Achievement Images

This directory contains image assets for achievements within the AUSTA SuperApp.

## Naming Convention

`achievement_[achievement_id].[file_extension]`

Example: `achievement_daily_steps.png`

## Storage

All achievement images should be stored in this directory.

## Usage

Reference these images within the `AchievementBadge` component or `AchievementNotification` component to visually represent user accomplishments.

Ensure that images are optimized for different screen sizes and resolutions to maintain a consistent user experience.
