# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability, please report it to security@austa.com.br. We will respond as quickly as possible.

## Supported Versions

Only the latest version of the AUSTA SuperApp receives security updates. Older versions are not supported.

## Security Practices

### Data Encryption

All sensitive data is encrypted at rest and in transit using industry-standard encryption algorithms (AES-256, TLS 1.3).

### Access Control

Access to sensitive resources is strictly controlled using role-based access control (RBAC) and multi-factor authentication (MFA).

### Regular Security Audits

The AUSTA SuperApp undergoes regular security audits by independent security experts.

### Vulnerability Scanning

Automated vulnerability scanning is performed regularly to identify and address potential security weaknesses.

### Incident Response

A well-defined incident response plan is in place to handle security incidents effectively.
